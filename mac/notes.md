# 📦 更新日志

### ✨ 新增功能
- **macOS 平台原生支持**：采用 Avalonia UI 跨平台框架，开发了与 Windows 版视觉效果和功能 1:1 一致的 macOS 桌面 GUI 客户端。
- **macOS 原生状态栏托盘**：适配 macOS 系统右上角状态栏，支持隐藏/显示主窗口、启停监控、配置自启及一键退出。
- **macOS 开机自启动**：支持在 `~/Library/LaunchAgents` 下自动写入 plist 启动配置文件，实现 macOS 下原生的开机静默自启。

### ⚡ 性能优化与体验改进
- **磁盘监控零卡顿 (macOS)**：引入内存级磁盘元数据缓存机制，每个 U 盘在挂载周期内仅调用一次 `diskutil`。同时将扫描工作移至后台 `Task.Run` 执行，彻底消除了界面线程阻塞与卡顿，CPU 占用降至约 0%。
- **Mac 专属系统文件过滤**：在文件复制逻辑中，自动跳过并排除 `.DS_Store`、`.Trashes` 以及 `._` 前缀等 Mac 专有系统冗余文件，确保备份出的课件干净利落。

### 🛠️ 编译与发布升级
- **双平台单文件发布**：支持一键编译生成 Windows x64 自包含单文件（`Rick.exe`，免安装 .NET 运行环境）与 macOS arm64 专属应用包（`获取Rick课件.app`）。
- **版本号规范化**：全平台工程配置、窗口标题及打包元数据版本号统一升级至 `v2.5`。
- **Finder 图标集成**：macOS `.app` 程序集成高质量 `.icns` 拟物化图标，Finder 界面无缝呈现。
