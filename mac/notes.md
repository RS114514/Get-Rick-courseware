# 📦 更新日志

### ✨ 新增功能
- **跨平台多操作系统支持（新）**：采用 Avalonia UI 跨平台框架，开发了与 Windows 版视觉效果和功能 1:1 一致的 macOS 桌面 GUI 客户端。
- **macOS 原生状态栏托盘（新）**：功能适配 macOS 系统右上角状态栏，支持隐藏/显示主窗口、启停监控、配置自启及一键退出。
- **macOS 开机自启动（新）**：支持在 `~/Library/LaunchAgents` 下自动写入 plist 启动配置文件，实现 macOS 下原生的开机静默自启。
- **U盘屏蔽与精准识别**：支持在界面下拉列表中直接屏蔽指定 U 盘，插入后自动跳过不复制。支持同名 U 盘通过序列号/UUID 精准识别。
- **运行日志持久化**：运行日志自动保存至程序目录下 `logs/` 文件夹，按天分文件，重启不丢失。
- **防重复启动限制**：程序已在运行时再次启动会进行友好提示并直接退出，避免多实例发生冲突。

### ⚡ 性能优化与体验改进
- **磁盘监控零卡顿 (macOS)（新）**：引入内存级磁盘元数据缓存机制，每个 U 盘在挂载周期内仅调用一次 `diskutil`。同时将扫描工作移至后台 `Task.Run` 执行，彻底消除了界面线程阻塞与卡顿，CPU 占用降至约 0%。
- **Mac 专属系统文件过滤（新）**：在文件复制逻辑中，自动跳过并排除 `.DS_Store`、`.Trashes` 以及 `._` 前缀等 Mac 专有系统冗余文件，确保备份出的课件干净利落。

### 🐛 问题修复
- **Windows 开机闪退修复**：修复了 Windows 下开机自启时程序启动即发生闪退的问题。
- **Windows 开机自启失效**：修复了开机自启配置无法固化到本地文件的问题，状态永久保存。
- **文件属性版本号**：修复了编译后 exe/dll 文件属性中版本号为空或不匹配的问题。

### 🛠️ 编译与发布升级
- **双平台单文件发布**：支持一键编译生成 Windows x64 自包含单文件（`Rick.exe`，免安装 .NET 运行时）与 macOS arm64 专属应用包（`获取Rick课件.app`）。
- **版本号规范化**：全平台工程配置、窗口标题及打包元数据版本号统一升级至 `v2.5`。
- **Finder 图标集成**：macOS `.app` 程序集成高质量 `.icns` 拟物化图标，Finder 界面无缝呈现。
- **向下兼容**：保留原有课件拉取的全部核心功能，配置读取逻辑无破坏性改动，无缝兼容 `v1.1` 旧版配置文件数据。
